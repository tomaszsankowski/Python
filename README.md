## Setup and Requirements

- **Python**: Version 3.8 or higher is recommended.
- **Virtual Environment**: It is highly recommended to use `venv` or `conda` to manage dependencies.
- **Core Libraries**: Most projects rely on `NumPy`, `Pandas`, `Matplotlib`, and `PyTorch` (for NN/RL labs).

## How to Run

1. **Clone the repository** and navigate to the project of your choice.
2. **Create and activate a virtual environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. **Install dependencies** (if a requirements.txt is present):
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the script:**
   ```bash
   python main.py
   ```
